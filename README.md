# Sales Data Analysis Project

## Overview
This project analyzes retail sales data using Python, Pandas, Matplotlib, and Seaborn to identify business insights and sales trends.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
The dataset contains information related to:
- Orders
- Products
- Sales
- Profit
- Customers
- Regions

## Key Analysis Performed
- Data Cleaning
- Handling Missing Values
- Removing Duplicates
- Monthly Sales Analysis
- Category-wise Sales Analysis
- Region-wise Sales Analysis
- Top Selling Products
- Correlation Heatmap

## Visualizations
The project includes:
- Bar Charts
- Line Charts
- Heatmaps
- Sales Trend Analysis

## Files Included
- sales.csv → Original dataset
- cleaned_sales.csv → Cleaned dataset
- sale_project.ipynb → Jupyter notebook containing analysis

## Insights
- Technology category generated high sales revenue
- Sales trends varied across months and regions
- Certain products contributed significantly to overall sales
